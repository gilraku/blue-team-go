# port-scanner

Part of [blue-team-go](../README.md) — a collection of CLI security tools written in Go.

## Usage

```
go run ./cmd/port-scanner [flags]
```

Run with `-h` to see all available flags.

## Build

```bash
go build -o bin/port-scanner ./cmd/port-scanner
```
